# Mahabharata

A structured repository for collecting, organising, studying and preparing Mahabharata-related files.

This repository is intended to hold source material, notes, translations, references and derived outputs in a clean, traceable structure.

## Purpose

The aim is to create an organised working base for Mahabharata study and future content development.

This repo can support:

- original source files
- chapter or parva notes
- translation comparisons
- commentary and interpretation notes
- research references
- prepared outputs for articles, scripts, podcasts or publications

## Repository structure

```text
Mahabharata/
├── sources/          # Original or reference source files
├── notes/            # Working notes, summaries and observations
├── translations/     # Translation drafts or comparison files
├── references/       # Bibliography, links and citation notes
├── outputs/          # Finished or publishable outputs
└── archive/          # Superseded, historical or parked material
```

## File naming convention

Use simple, sortable names.

Recommended format:

```text
YYYY-MM-DD_topic_short-description.md
```

Examples:

```text
2026-07-07_bhishma_overview.md
2026-07-07_gita_notes_chapter-02.md
2026-07-07_translation_comparison_adiparva.md
```

## Working rules

- Keep original source files separate from edited notes.
- Do not overwrite source material.
- Put draft work in `notes/` or `translations/`.
- Put finished material in `outputs/`.
- Use `archive/` for old drafts rather than deleting them.
- Add a short note at the top of files explaining provenance where useful.

## Suggested file header

```markdown
# Title

Date: YYYY-MM-DD
Status: draft / review / final
Source: original / translated / commentary / derived
Notes:
```
